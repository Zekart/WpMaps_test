Меню Содержит в себе 3 подменю: Поиск - поиск адреса по карте, Легенда - открытие легенды, Подсказка - открытие окна помощи. Содержит в себе настройки, что касаються пользователя системы. LogOff - выход до окна авторизации. Среднее меню управления, содержит в себе 2 режима перехода: "basic" - только просмотр информации, "admin" - режим изменения данных Будет содержать натройки, что связанно с системой: отображение, планирование и тд. Будет содержать стандартное меню "Файл", с возможностью печати информации нужного объекта или же карту с маршрутами.. Управление *режим admin* Добавить объект Добавление объекта осуществляется с помощью клика по карте, в открытом окне, введите всю необходимую информацию. Добавление объекта в дальнейшем будет поделено на 2 режима "простой" - только объекты и "стандарт" - заполнения всей информации, что касается объектов (столбов, колодцев, домов). Отобразить объекты Отображение объектов просходит при приближении карты (увеличения зума). Информация объекта При нажатии на маркер происходит вывод краткой информации. Доступны не которые функции: "Удалить" - удаляет объект с карты (базы), "Информация" - выводит информацию о муфтах объекта, "Изменить" - диалоговое окно из изминениями информации, "Соединить" - соединяет 2 объекта (выбираем первый, нажали "соединить", переходим ко второму и нажимаем "соединить"). В дальнейшем, всё это будет автоматически. Удаление\Добавление "Соединений". Удаление не работает, в связи с тем, что будет в дальнейшем авт. прорисовка по информации с кабеля.
